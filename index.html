Projects

Tavern - Co creator

Tavern is a whisky encyclopedia where users can create create accounts, read, leave their own reviews, and follow other users of the app.

I'm the co creator and responsible for all the frontend. I used React, Apollojs, and graphQL to create the UI/UX. The styling was done using react-semantic-ui, bootstrap 4, and various libraries such as Chart.js


Aggo - Co Creator

Aggo is a productivity app for the iOS. It was created to be a life planner where the user can break down large goals into daily steps in a visual way. I made the frontend using React Native and mobX state tree


This or That - Solo developer

This or That is a social polling web app. By signing up users can create and vote on whatever poll they want. The results are displayed as a bar chart so they can easily be visualized. Users can also follow and see what their friends have posted in a feed.

This was my first fullstack web application. The stack I used was Postgres, Ruby on Rails, and JavaScript/jQuery.
